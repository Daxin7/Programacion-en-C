---

## 🛣️ Ruta de Aprendizaje

Esta es la estructura que estaré siguiendo:

1.  **Fundamentos:** Tipos de datos, operadores y entrada/salida de datos.
2.  **Estructuras de Control:** Condicionales (`if`, `switch`) y bucles (`for`, `while`, `do-while`).
3.  **Funciones:** Modularización del código y paso de parámetros.
4.  **Arreglos y Listas:** Vectores, matrices y manipulación de cadenas de caracteres.
5.  **Punteros:** Gestión de memoria, punteros a funciones y estructuras.
6.  **Estructuras y Archivos:** Creación de tipos de datos personalizados y persistencia de datos.

## 📝 Notas de Aprendizaje

### Cómo compilar mis archivos
Para ejecutar cualquiera de mis ejercicios, utilizo el siguiente comando en la terminal:

> ```bash
> gcc nombre_archivo.c -o ejecutable
> ./ejecutable
> ```
