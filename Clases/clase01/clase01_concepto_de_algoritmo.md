> [!NOTE]
> # ¿Qué es un algoritmo?
> * Un algoritmo es un método para resolver un problema
> * El algoritmo debe ser ordenado, finito y no debe tener ambiguedades


> [!TIP]
> ## Pasos para resolver un problema
> * Diseño del algoritmo
> * Expresar el algoritmo como un programa
> * Ejecución y validación del programa por la computadora

> [!NOTE]
> ## Cáracteristicas de los algoritmos
> * Un algoritmo debe ser preciso.
> * Un algoritmo debe estar definido.
> * Un algoritmo debe ser finito.
